一、專題摘要 
=============
1. 專題主題 : 線上發票辨識系統

2. 專題基本目標 : 
  * 使用者透過line bot上傳統一發票發票後，可以知道是否有中獎
  * 爬取財政部的統一發票中獎資料存入AWS的MongoDB server。
  * 透過Azure表格辨識器訓練統一發票的圖片。
  * 撰寫統一發票中獎與金額的邏輯定義。
